# Event Management System - DBMS Project

This repository contains a DBMS course project based on an **Event Management System**.  
The project focuses on designing a database system for managing events, users, venues, catering, decorations, and programmes.

## Project Overview

The Event Management System is designed to store and manage information related to different events.  
Each event has an event ID, type, date, and related user information. The system also stores details about venue, catering, decoration, and programme arrangements for each event.

## Main Entities

- Event
- Users
- Venue
- Catering
- Decoration
- Programme

## Note

This project was completed as part of the **Introduction to Database** course at **American International University - Bangladesh**.  
It is a DBMS course project created to demonstrate database design, ER modeling, normalization, table creation, data insertion, SQL query writing, joins, views, and relational algebra.

This repository is intended to show the project work and learning outcomes.

## Conclusion

The Event Management System project shows how database concepts can be used to organize and manage event-related information. The system stores event details, user records, venue information, catering details, decoration costs, and programme schedules.

Through this project, we practiced designing an ER diagram, normalizing database tables, creating tables using SQL, inserting sample data, writing queries, using joins, creating views, and applying relational algebra.

Overall, the project helps simplify the process of storing and retrieving event management data in a structured database system.
